# EV Pao Mobile App

This repo is designed to connect to the ev pao project and display data from the device.

prerequisites:
 - node
 - yarn
 - expo-cli
 - expo

instructions:

`$ yarn install`

`$ yarn start`

select appropriate option to run on simulator or device
